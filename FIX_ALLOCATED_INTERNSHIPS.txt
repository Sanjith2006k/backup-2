What it does:
  1. Takes 90% of students
  2. Finds their BEST matching internship
  3. Sets StudentProfile.allocated_internship = internship title
  4. Sets StudentProfile.match_score = match quality score
  5. Updates application status to 'approved'

Code Added:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```python
for student_data in dataset['students']:
    # 90% of students get allocated
    if random.random() < 0.90:
        # Find best matching internship
        best_match = find_best_match(student_data, internships)

        # Allocate
        student_profile.allocated_internship = best_match['title']
        student_profile.match_score = match_score
```
